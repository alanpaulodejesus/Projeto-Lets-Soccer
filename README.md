# 🚀⚽🏃‍♂️ Projeto-Lets-Soccer

Lets Soccer é um projeto para amantes de futebol, afinal que não gostaria de dar "pitaco" na escalação do seu time?
Aqui o torcedor poderá escalar o seu time ideal do clube do coração, antes de cada partida.
Ao final de cada votação do time ideal, será disponibilizado os jogadores mais votados para aquele confronto.

E que comecem os jogos!!

---

## ✅ Pré-requisitos

Antes de executar o projeto, certifique-se de ter os seguintes itens instalados na sua máquina:

- Java (versão 17 ou superior recomendada)
- Maven

---

## ⚙️ Funcionalidades:

- Login usuário Admin (Permissão de cadastro, votação e consulta) 
- Login usuário (Permissão de votação e consulta)
- Crud de jogadores (Acesso Admin)
- Votação de jogadores
- Consulta de jogadores mais votados
